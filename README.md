## Challenges Encountered and Solutions

### 1. Schema Validation

Validated each schema using Schema.org tools and Google’s Rich Results Test. Ensured the same logo, sameAs links, and @id values were consistent across all entity types.

### 2. Page Speed and Core Web Vitals

Used compressed images, and a light theme to improve speed. This also reduced layout shift and improved performance on mobile.

### 3. Entity Identity Alignment

Made sure the Organization, Person, and Website schemas aligned in terms of URL structure, logo usage, and branding signals.
